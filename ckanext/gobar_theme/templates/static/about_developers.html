The catalog

Our portal uses the CKAN platform version 2.5. CKAN is a tool developed by the Open Knowledge Foundation. This is free software distributed under the GNU Affero General Public License.

Currently it is one of the most popular platforms for this type of initiatives. It is used by, among others, the Government of the United Kingdom, the United States, Brazil and Chile.

CKAN allows the organization of published data through its dataset and resource schema, as well as programmatic access to them, applying internationally approved standards for metadata generation.

Metadata

To make it easier to navigate the portal and search datasets, we use the metadata standards proposed by default in CKAN (dublin core and dcat).

Integrate our data

You can integrate the data and metadata of a dataset by means of the CKAN API.

Here is the generic way to query metadata: 

http://data.soscbaha.org/api/3/action/package_show?id=<id_del_dataset>

For example, we could see the metadata of the Organic Structure dataset and authorities of the National Executive Power, consulting: 

http://data.soscbaha.org/api/3/action/package_show?id=estructura-organica-pen

This is the generic way to query data for a resource within of a specific dataset: 

http://datos.gob.ar/api/3/action/datastore_search?resource_id=<id_del_recurso>

An example? We can see the data of a resource of the Organic Structure dataset and authorities of the National Executive, as follows: 

http://data.soscbaha.org/api/3/action/datastore_search?resource_id=b705d8c1-650f-43cc-bde6-68850fcecd21

See more details for using the API.
